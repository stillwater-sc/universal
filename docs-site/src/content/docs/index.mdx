## Quick Example

```cpp
#include <universal/number/cfloat/cfloat.hpp>

template<typename Real>
Real MyKernel(const Real& a, const Real& b) {
    return a * b;
}

int main() {
    using Real = sw::universal::half;  // half-precision IEEE-754
    Real a = sqrt(2);
    Real b = 3.14159265358979;
    std::cout << "Result: " << MyKernel(a, b) << std::endl;
}
```

## Number Systems at a Glance

| Category | Types |
|----------|-------|
| **Integer & Fixed-Point** | `integer`, `fixpnt`, `rational` |
| **Configurable Float** | `cfloat`, `bfloat16`, `areal`, `dfloat` |
| **AI Quantization** | `microfloat`, `e8m0`, `mxfloat`, `nvblock` |
| **Posit Family** | `posit`, `quire`, `takum` |
| **Logarithmic** | `lns`, `dbns` |
| **Interval** | `valid`, `interval`, `sorn` |
| **Extended Precision** | `dd`, `qd`, `dd_cascade`, `td_cascade`, `qd_cascade` |
