description: How to cite the Universal Numbers Library in academic work
---

Please cite [our work](https://joss.theoj.org/papers/10.21105/joss.05072) if you use the Universal Numbers Library.

## Published Papers

1. [Universal Numbers Library: Multi-format Variable Precision Arithmetic Library](https://joss.theoj.org/papers/10.21105/joss.05072) — JOSS 2023
2. [Universal: Reliable, Reproducible, and Energy-Efficient Numerics](https://link.springer.com/chapter/10.1007/978-3-031-09779-9_7) — CoNGA 2022
3. [Iterative Refinement with Low-Precision Posits](https://arxiv.org/abs/2408.13400) — arXiv 2024
4. [Reproducibility](https://arxiv.org/abs/2012.11011) — arXiv 2020

  volume={8},
  number={83},
  pages={5072},
  year={2023}
}
